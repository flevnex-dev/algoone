# AlgoOne Website Clone

This is a pixel-perfect clone of the AlgoOne dashboard website (https://algoone-dashboard.com/) built with Tailwind CSS.

## Features

- ✅ Complete website clone with all sections
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Modern UI with Tailwind CSS
- ✅ All original sections included:
  - Top banner
  - Header with navigation
  - Hero section
  - Elite Trading Signals section
  - How It Works section
  - Proven Track Record section
  - Why Choose AlgoOne section
  - Referral Program section
  - Ready to Start Trading section
  - Footer

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome Icons
- Custom CSS for enhanced styling

## How to Use

1. Simply open `index.html` in your web browser
2. No build process or installation required
3. All assets are loaded via CDN

## File Structure
